# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1][covenant], a
widely-used code of conduct for open-source projects. Maintainers and
contributors are expected to uphold the standards it describes.

## Reporting

To report unacceptable behavior, email `statespaceecon@gmail.com`. All
reports will be reviewed and investigated promptly and fairly. Reporter
identity will be kept confidential.

## Enforcement

Maintainers may take any action they deem appropriate, up to and
including a permanent ban from the project's communication channels, in
response to behavior they deem inappropriate, threatening, offensive, or
harmful. See the Covenant's [Enforcement Guidelines][guidelines] for the
standard four-tier response framework.

[covenant]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
[guidelines]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines
