# TimeSeriesEconPy

A time-series language for macroeconomics, ported from
[TimeSeriesEcon.jl](https://github.com/bankofcanada/TimeSeriesEcon.jl).

!!! warning "Pre-alpha"
    The package is in M0 (repo skeleton). No public API yet. This site exists
    so the build pipeline is wired up; real content arrives during M1.

## Quick links

- [GitHub repository](https://github.com/Nic2020/TimeSeriesEconPy)
- [TimeSeriesEcon.jl (Julia upstream)](https://github.com/bankofcanada/TimeSeriesEcon.jl)
- [Issues](https://github.com/Nic2020/TimeSeriesEconPy/issues)

## What this site will contain (M1+)

- **Getting started** — install, first TSeries, plot, save.
- **Concepts** — frequencies, MIT, TSeries, MVTSeries, Workspace.
- **Guides** — frequency conversion, pandas/polars interop, plotting backends.
- **API reference** — auto-generated from docstrings.
- **Migration** — coming from pandas, coming from TimeSeriesEcon.jl.
